# Coffee Catalog

8 specialty coffees, each with multiple variants (form x size).

## Variant System

Each coffee has the following variants:

| Form | Sizes | Price modifier |
|---|---|---|
| Whole bean | 250g / 500g / 1kg | base / +surcharge / +surcharge |
| Ground — filter | 250g / 500g | +0 Ft |
| Ground — espresso | 250g / 500g | +0 Ft |
| Ground — french press | 250g / 500g | +0 Ft |
| Ground — turkish | 250g | +0 Ft |
| Drip bag | 10 pcs / 20 pcs | base+500 / base×2+500 |

SKU formula: `CB-{ORIGIN_CODE}-{FORM}-{SIZE}` (e.g. `CB-ETH-WB-250`, `CB-COL-GF-500`)
